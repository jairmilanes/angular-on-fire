Angular On Fire

Pre-configured Angular, Firebase, Cypress, Jest and server side rendering application

Packet With Power

Angular on Fire comes packet with the best tools in the Market, all pre-configured and ready to help you get started on your projects faster.

Angular 8

All the latest features like web workers, deferential and improved lazy loading, Node 10 support and all the goodies Angular provides.

Firebase

The power of Firebase services at your disposal, file storage, object database, hosting, cloud functions and much more.

Cypress

Cypress has taken the development community by storm, with it's great interface and ease to use apis, you can have your application tested in no time.

Jest

Test runner created by Facebook, widely used, vast documentation and community support.

Usage

Start by forking your own version of Angular on Fire and clone the repository to your local machine.

Follow the steps in the README.md to get your project up and running in only a couple of minutes.

Ci/Cd Ready

Angular on Fire comes ready for ci/cd deployment with Circle CI

To get started with builds follow these steps:

  • If you don't already have one, you can get a free account to get started here, you can log in with your github account to make it even easier to configure.
  • Follow the steps to connect your Github account and link your project to Circle Ci.
  • Make sure to skip file configuration steps since everything is already configured for you, all you need is to link your project to Circle Ci.
  • Then just push code to your repository, a siple commit is enough to trigger your build.
